# 封面模式输出模板

严格使用以下结构：

```markdown
# 图片目标

一句话说明封面在文章中的作用，不复述标题。

---

# 核心观点

一句话表达真正需要传达的关系、变化或因果；去掉品牌名后仍成立。

---

# 视觉隐喻

说明选择的现实场景或必要的实物隐喻，以及它如何对应核心观点。真实场景能直接表达时，
不要强行制造象征。

---

# 构图

- 主体：
- 动作：
- 镜头：
- 背景：
- 留白：
- 焦点：

---

# 风格

说明编辑气质、真实材质、色彩与可解释光线。

---

# Prompt

目标模型：

最终 Prompt：

长度：中文 50–60 个汉字；英文 35–45 个单词。

只写真实场景、主体、动作、核心关系、光线、色调与必要留白。

---

# Negative Prompt

针对当前方案列出需要避免的元素。

---

# 自检

- 表达观点而非新闻：
- 单一焦点：
- 容易理解：
- 符合科技媒体：
- 避免常见撞图套路：
- 场景、动作和光线明确：
- Prompt 长度符合目标：
- 综合评分：X/10
```
