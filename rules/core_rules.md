# 核心推理规则

## 从事件到观点

先分别写出：

- **事件层**：文章描述了什么变化或事实。
- **观点层**：作者认为这件事意味着什么。
- **读者层**：读者看完后应理解什么。

最终图片只服务于观点层和读者层。

## 核心观点测试

合格的核心观点应：

1. 能写成一句话。
2. 去掉品牌名后仍然成立。
3. 包含一个明确关系或变化。
4. 不使用“以及”“同时”“并且”连接第二主题。

## 视觉方案测试

在写 Prompt 前，用一句话完成：

`通过【一个主体】发生【一个动作或状态变化】，表达【核心观点】。`

无法填入一个主体和一个动作时，继续简化。

## 错误示例

文章：某 AI 产品取消固定五小时限制。

- 错误：产品 Logo、时钟、机器人、代码界面和欢呼的人群同时出现。
- 原因：这是新闻关键词集合，没有表达限制机制发生了什么变化。
- 正确方向：让被时间刻度切断的工作流转化为可连续分配的资源轨道。
