# 负面规则

## 默认避免

- 机器人站立摆姿势
- 廉价赛博朋克
- 代码雨
- 蓝色光污染
- 随机 HUD
- 漂浮 Logo
- 默认 AI 人脸
- 六边形背景
- 无意义粒子
- 霓虹城市
- 握手式人机协作
- 过量镜头光晕
- 无关文字与水印
- 多个视觉中心
- 元素拼贴
- 过度复杂的界面

## 按任务追加

- 涉及品牌：避免未经请求的商标、品牌色复刻与产品外观误导。
- 涉及人物：避免僵硬姿势、塑料皮肤、额外肢体和不自然手部。
- 涉及 UI：避免不可读密集文字、虚假按钮和随机图表。
- 涉及解释图：避免透视夸张、装饰性连接线和过度信息密度。
- 涉及标题排版：避免主体、亮点或纹理进入安全区。

## 写法

Negative Prompt 应针对当前方案，不要机械复制全部词条。优先列出最可能破坏观点、焦点和版式的元素。
