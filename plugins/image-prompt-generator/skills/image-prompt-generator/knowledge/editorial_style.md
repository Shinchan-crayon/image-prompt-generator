# 科技编辑视觉质量

## 何时读取

核心场景和构图已经确定，需要校准材质、信息密度、色彩、光线与编辑气质时读取。
本文件只提供可组合的质量属性，不提供品牌复刻模板。

## 使用边界

Apple、OpenAI、Anthropic、Nothing、MIT Technology Review、The Verge、Bloomberg、
WIRED 只作为内部质量索引。以下属性是对常见视觉特征的抽象归纳，不是品牌规范，
也不代表所有时期、栏目或作品。

不得：

- 在 Prompt 中直接要求“模仿某品牌或媒体风格”；
- 复制 Logo、品牌色、字体、版式、产品外观或标志性作品；
- 把多个参考名称堆在一起制造空泛“高级感”；
- 让风格覆盖文章事实、核心关系和真实场景。

正确用法是从下列索引中选择两到四项具体属性，再翻译为材质、镜头、光线和信息密度。

## 快速质量轴

| 质量轴 | 低值 | 高值 | 选择依据 |
|---|---|---|---|
| 信息密度 | 单物体、宽留白 | 多证据、紧凑层级 | 封面偏低，分析型正文可适度提高 |
| 现实感 | 棚拍、受控表面 | 纪实、使用痕迹 | 产品审美偏前者，社会与行业议题偏后者 |
| 情绪强度 | 平静、均匀光 | 强对比、紧迫色 | 服从文章证据，不服从新闻热度 |
| 材质强调 | 光滑、统一 | 纸张、金属、织物、磨损 | 需要可信操作时提高材质可见度 |
| 色彩对比 | 中性克制 | 明确撞色 | 观点冲突或文化议题可提高 |
| 版式秩序 | 对齐、规则网格 | 拼贴、张力、断点 | 解释效率偏规则，观点表达可适度增加张力 |

## 抽象质量索引

### 01. Apple：产品秩序与材质精度

- **可提取属性**：单一焦点、精确边缘、充足负空间、受控反射、材质真实、动作简洁。
- **转译方式**：减少背景物件，用近景或中近景展示真实产品在一个明确动作中的触感与尺度。
- **适用场景**：产品测评、工业设计、硬件体验、工具简化。
- **避免**：复制发布会白底、设备悬浮、品牌化产品阵列、把生活议题拍成奢侈品广告。

### 02. OpenAI：清晰系统与人的使用关系

- **可提取属性**：开放空间、柔和中性色、界面克制、人的动作清楚、复杂能力被简化为可理解交互。
- **转译方式**：让人物、真实屏幕和任务结果形成直接关系，界面只保留与观点有关的状态。
- **适用场景**：AI 产品、Agent、办公工具、模型能力进入日常工作。
- **避免**：复制品牌图形、抽象黑白纹理、空泛未来空间、用聊天框代替所有 AI 议题。

### 03. Anthropic：研究感与审慎边界

- **可提取属性**：纸张与档案质感、低饱和暖色、思考痕迹、证据并置、克制而有人文尺度。
- **转译方式**：使用研究桌、批注、审核动作和自然材料表现复杂判断与责任边界。
- **适用场景**：安全、政策、研究、模型边界、长期影响。
- **避免**：复刻品牌暖色、把所有画面做成复古文档、用书本堆叠代替真实证据。

### 04. Nothing：结构可见与工业个性

- **可提取属性**：结构层次、透明或半透明材质、黑白基础色、小面积高识别强调色、精密但不繁复。
- **转译方式**：在产品或机制确实需要展示内部关系时，用真实剖面、装配细节和受控光线揭示结构。
- **适用场景**：工业设计、硬件结构、产品差异化、制造。
- **避免**：无理由透明外壳、点阵字体复刻、黑红模板、把软件或医疗主题强行机械化。

### 05. MIT Technology Review：议题概念与编辑克制

- **可提取属性**：观点优先、单一强隐喻、科学或社会尺度、有限色彩、画面允许不舒适的问题意识。
- **转译方式**：用一个现实场景中的异常关系提出问题，保持证据边界，不依赖视觉奇观。
- **适用场景**：技术社会影响、政策、风险、研究突破、长期趋势。
- **避免**：复制杂志封面结构、把复杂议题缩成聪明符号、为“思想性”制造晦涩。

### 06. The Verge：文化张力与鲜明层级

- **可提取属性**：强主次、清晰轮廓、有限撞色、消费科技现场感、适度活力与文化语境。
- **转译方式**：用一个高识别现实物件和明确动作建立焦点，以小面积色彩强调冲突或变化。
- **适用场景**：消费科技、平台文化、产品竞争、数字生活。
- **避免**：复制标志性渐变、夸张拼贴、每篇都用高饱和霓虹、让文化感压过信息。

### 07. Bloomberg：证据密度与商业现场

- **可提取属性**：事实感、数据意识、真实办公或产业环境、紧凑构图、冷静中性色与明确强调色。
- **转译方式**：让订单、排期、产能、价格或选择动作在真实业务现场中可见，不编造具体数字。
- **适用场景**：商业竞争、融资、供应链、市场变化、公司分析。
- **避免**：虚假仪表盘、股价箭头、交易大厅模板、用金钱符号代替商业机制。

### 08. WIRED：技术文化与受控实验性

- **可提取属性**：大胆单概念、人物或物体的陌生化视角、颜色张力、精确棚拍、技术与文化并置。
- **转译方式**：只在观点需要挑战常规视角时，改变尺度、取景或材质组合，并保持一个可解释关系。
- **适用场景**：未来趋势、技术文化、身份、创意工具、争议性产品。
- **避免**：复制人物肖像套路、无意义怪异化、超现实元素过多、实验性破坏三秒理解。

## 属性组合

每次最多选择：

1. 一个**信息密度**属性；
2. 一个**现实或材质**属性；
3. 一个**色彩或光线**属性；
4. 必要时一个**情绪张力**属性。

示例：

`低信息密度 + 真实使用痕迹 + 中性色与小面积橙色 + 冷静审慎`

应翻译为：

`真实办公室中景，磨损木桌与纸质审核记录，中性窗光，异常位置用小面积橙色标记。`

不要写成：

`Apple、Anthropic、Bloomberg 混合风格，高级科技感。`

## 通用品质规则

- 场景优先真实办公室、商店、家庭、教室、医院、道路、工厂或桌面。
- 材质优先纸张、木材、金属、织物、玻璃和有使用痕迹的真实屏幕。
- 色彩以中性色建立秩序，只用一到两种强调色表达关系。
- 光线优先自然窗光、现场工作光或可解释的棚拍光，不使用无来源发光。
- UI 只在交互本身是观点核心时出现，并保持真实、低噪音、可核对。
- 人物只在选择、体验、责任或协作是观点核心时出现；动作比表情重要。

## 风格失败时

- **像品牌广告**：移除品牌中心、完美表面和英雄产品角度，加入现实任务与使用痕迹。
- **像杂志仿作**：删除品牌名称，重新写成镜头、材质、密度、光线和色彩。
- **风格盖过观点**：保留证明核心关系的动作，降低色彩、特效和陌生化程度。
- **同类文章撞图**：更换现实地点、工件或动作，不只替换强调色。
