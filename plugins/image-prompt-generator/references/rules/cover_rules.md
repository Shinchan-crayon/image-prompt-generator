# 封面模式规则

## 目标

封面应在三秒内传达文章的核心意义，并具备专业科技媒体的编辑品质。

## 必须做到

- 使用完整文章判断作者真正的结论。
- 选择一个视觉隐喻、一个主要场景、一个视觉焦点。
- 优先设计现实中可拍摄的生活或工作场景。
- 让人物或实物发生一个明确动作，而不是静态物件陈列。
- 明确自然光线或可解释的现场光线。
- 指定标题安全区的位置、面积与低干扰要求。
- 根据比例调整主体位置与空间分配。
- 风格描述应说明材质、色彩、光线和编辑气质。

## 不得采用

- 只画公司 Logo、产品截图或新闻人物。
- 把多个新闻要点做成拼贴。
- 让标题安全区与关键叙事元素重叠。
- 使用电影海报式爆炸、英雄姿态或夸张灾难场景。
- 用“未来感、科技感、高级感”代替具体视觉设计。
- 用抽象工业装置代替本可生活化表达的真实场景。

## 设计顺序

1. 一句话核心观点。
2. 一个视觉隐喻。
3. 一个可观察动作。
4. 一个主体焦点。
5. 一个明确留白区。
6. 风格与光线。
7. 压缩为符合模型长度预算的最终 Prompt。
