# 色彩语言

## 何时读取

核心观点、主情绪和现实场景已经确定，需要选择基础色与强调色时读取。本文件只调整视觉
语气，不用色彩替代观点或改变事实强度。

色彩用于强化文章关系和情绪，不用于替代观点。

## 基础语义

| 色彩 | 常见语义 | 适用方向 | 风险 |
|---|---|---|---|
| 蓝色 | 理性、系统、可信 | 基础设施、企业技术、数据治理 | 容易变成蓝色光污染 |
| 红色 | 警告、冲突、紧迫 | 风险、安全、竞争、政策冲击 | 容易制造夸张恐慌 |
| 绿色 | 增长、效率、恢复 | 商业增长、节能、长期价值 | 容易像金融广告 |
| 黑色 | 克制、精密、高端 | 工业设计、芯片、产品测评 | 容易丢失生活感和细节 |
| 白色 | 清晰、开放、轻量 | 产品、教育、工具简化 | 容易空洞或像电商白底图 |
| 黄色 | 注意、能量、启发 | 教育、速度、关键提醒 | 大面积使用会显得廉价 |
| 橙色 | 行动、温度、创造 | 创业、协作、生活化技术 | 容易变成营销海报 |
| 紫色 | 探索、未知、生成式 AI | 未来趋势、创意工具 | 已成为高频 AI 套路 |
| 灰色 | 中立、秩序、基础 | 行业分析、企业流程、政策 | 缺少强调色时显得沉闷 |

## 配色方法

1. 使用一个基础中性色建立真实空间。
2. 使用一个主要强调色表达关系或状态。
3. 必要时加入一个低占比辅助色，不形成彩虹式界面。
4. 让色彩来自真实物件、屏幕状态、纸卡或环境光，而不是无来源发光。

## 情绪联动

- 突破、成长：自然暖光与克制绿色或金色重点。
- 竞争、速度：高对比中性色与小面积红色或橙色。
- 风险、安全：低饱和中性色与受控红色，避免恐怖片照明。
- 教育、解释：明亮自然光、纸张和木材本色，强调色用于关系。
- 未来、预测：冷暖并置表达未知与可达性，不默认紫蓝渐变。

## 禁止简化

不要机械套用“蓝色等于科技、红色等于危险、黑色等于高级”。若色彩与文章情绪、
真实场景或品牌事实冲突，以文章语义和现实可信度为准。
