# 作者意图库

## 何时读取

文章主题与主分类确定后读取。写作意图回答“作者希望读者如何理解或行动”，并决定画面要呈现变化、证据、机制、步骤、体验还是趋势。

先选择一个主意图。只有第二意图会改变画面任务时才记录辅助意图，不为每个意图分别设计画面。

## 快速索引

| 主要文本行为 | 意图 |
|---|---|
| 告知刚发生的变化 | 新闻 |
| 说服读者接受一个判断 | 观点 |
| 拆解原因、结构或影响 | 分析 |
| 揭开被隐藏或被忽略的机制 | 揭秘 |
| 教读者完成任务 | 教程 |
| 用真实过程证明方法 | 案例 |
| 通过受访者经验理解问题 | 采访 |
| 用实际使用判断产品 | 评测 |
| 推演尚未发生的结果 | 预测 |
| 解释收入、成本或战略 | 商业 |
| 促使读者关注、试用或购买 | 营销 |
| 宣布新产品或新能力 | 产品发布 |
| 持续观察行业共同变化 | 行业观察 |

## 01. 新闻

- **判断线索**：时间、事件、发布、变化和直接事实占主导，作者解释较少。
- **视觉任务**：表现“什么发生了改变”，不复刻发布会现场。
- **主体与动作**：一个受影响的人或实物正在适应新状态，例如用户开始使用新增能力。
- **构图选择**：单场景状态变化；封面保留标题区，正文聚焦变化发生的位置。
- **应避免的误读**：把标题判断当事实、堆公司 Logo、画成新闻现场照片或夸大影响范围。

## 02. 观点

- **判断线索**：明确判断句、价值选择、反驳或“真正重要的是”反复出现。
- **视觉任务**：让读者一眼看到作者主张的关系，而不是看到相关名词。
- **主体与动作**：一个主体做出选择、转向或重新分配注意力。
- **构图选择**：非对称单焦点构图，用方向、距离或资源移动表达立场。
- **应避免的误读**：把观点画成事实、同时容纳正反双方全部论据、用大字口号代替视觉关系。

## 03. 分析

- **判断线索**：原因、机制、数据、对比、限制和影响链条占主导。
- **视觉任务**：解释一个关键机制或因果，不总结所有章节。
- **主体与动作**：人物检查系统、实物沿路径移动、一个瓶颈限制整体流程。
- **构图选择**：俯视关系、剖面或中景操作，概念节点不超过四个。
- **应避免的误读**：复杂信息图、随机箭头、把相关性画成因果、把分析做成戏剧海报。

## 04. 揭秘

- **判断线索**：隐藏成本、幕后流程、未公开约束、表象与实际之间的差异。
- **视觉任务**：揭示可验证的内部结构或被遮挡关系，不制造阴谋感。
- **主体与动作**：一双手打开设备、翻开记录、移开遮挡物或检查后台流程。
- **构图选择**：近景、局部剖面或由前景遮挡到内部焦点的层次构图。
- **应避免的误读**：黑暗房间、神秘黑影、红线墙、没有证据却暗示欺骗或犯罪。

## 05. 教程

- **判断线索**：步骤、操作、前置条件、注意事项和完成标准明确。
- **视觉任务**：让读者看懂最关键动作和结果。
- **主体与动作**：人物或手部正在执行一个具体步骤，工具和结果可见。
- **构图选择**：俯视操作台、肩后视角或三到四阶段的单向路径。
- **应避免的误读**：只画成果、步骤过多、依赖长文字、把学习过程做成产品广告。

## 06. 案例

- **判断线索**：具体人物、团队或企业从问题到行动再到结果的过程。
- **视觉任务**：用一个关键决策或转折代表案例为何有效或失败。
- **主体与动作**：真实角色操作原型、调整流程、检查结果或处理限制。
- **构图选择**：纪实中景，保留环境证据；需要前后对照时保持同一视角和主体。
- **应避免的误读**：把单个案例推广为普遍结论、只画成功庆祝、删除失败条件。

## 07. 采访

- **判断线索**：问题与回答、个人经验、立场、观察和原话是主要证据。
- **视觉任务**：呈现受访者与其工作、选择或环境的真实关系。
- **主体与动作**：受访者正在工作、演示或查看关键对象，而非仅正面坐姿。
- **构图选择**：环境肖像或工作现场中景，人物为焦点，背景提供有限语境。
- **应避免的误读**：名人海报、舞台聚光、把个人观点当行业事实、无关的麦克风特写。

## 08. 评测

- **判断线索**：测试条件、使用体验、优缺点、对比对象和适用人群。
- **视觉任务**：展示一个决定结论的实际测试或限制。
- **主体与动作**：用户在同一任务下操作产品，结果或困难可见。
- **构图选择**：桌面近景、肩后视角或控制变量一致的对照构图。
- **应避免的误读**：发布会广告图、只画产品外观、比较条件不一致、把主观体验写成绝对性能。

## 09. 预测

- **判断线索**：未来、可能、预计、如果、情景推演和不确定条件。
- **视觉任务**：从现实信号展示一个正在形成的方向，并保留未完成感。
- **主体与动作**：现实人物开始调整准备，原型进入早期使用，路径逐渐形成。
- **构图选择**：清晰现实前景加有限延伸路径，不把未来结果全部实体化。
- **应避免的误读**：未来城市、传送门、把可能性画成定局、忽略预测条件。

## 10. 商业

- **判断线索**：收入、成本、客户、渠道、定价、竞争和资源配置。
- **视觉任务**：解释价值如何交换、成本落在哪里或战略如何改变。
- **主体与动作**：客户购买、员工交付、管理者分配资源或团队调整流程。
- **构图选择**：一次交易或一次资源取舍为焦点，辅助物件只解释机制。
- **应避免的误读**：金币和上升箭头、把营收等同利润、忽略成本与执行条件。

## 11. 营销

- **判断线索**：利益点、行动号召、品牌承诺、试用购买或线索转化。
- **视觉任务**：展示目标用户获得的具体使用结果，同时识别并限制宣传夸张。
- **主体与动作**：目标用户在真实场景中完成一个任务，产品处于辅助位置。
- **构图选择**：清晰产品使用场景和可用留白，焦点落在结果而非口号。
- **应避免的误读**：把广告主张标为事实、过度理想化人物、堆功能图标、复制品牌广告资产。

## 12. 产品发布

- **判断线索**：新品、新功能、版本更新、发布日期和官方能力说明。
- **视觉任务**：表现新能力改变了什么具体行为，不复刻舞台与包装。
- **主体与动作**：首批用户、开发者或实物正在使用一个新增能力。
- **构图选择**：单一产品行为近景或中景，封面使用编辑留白，正文突出交互位置。
- **应避免的误读**：发布会舞台、Logo 中心构图、功能全家桶、把官方声称当作独立验证。

## 13. 行业观察

- **判断线索**：多次事件、多个参与者、长期变化和阶段性信号被放在一起讨论。
- **视觉任务**：展示共同力量如何让行业参与者汇聚、分化或改变方向。
- **主体与动作**：同一现实系统中的多个对象受到一条共同规则或资源流影响。
- **构图选择**：有主关系的俯视或较宽中景，最多四个节点，避免 Logo 拼贴。
- **应避免的误读**：把零散新闻当趋势、无依据预测赢家、用地图箭头制造虚假规模感。

## 组合规则

- `新闻 + 观点`：画面服务观点，但证据强度保持新闻事实边界。
- `产品发布 + 评测`：若文章有独立测试，以评测为主；只有官方说明时以产品发布为主。
- `分析 + 揭秘`：有隐藏证据或表象反差才使用揭秘，否则使用分析。
- `教程 + 案例`：读者要照做时选教程；重点是证明方法有效时选案例。
- `预测 + 行业观察`：文章基于多次现实信号时选行业观察；主要推演未发生结果时选预测。
