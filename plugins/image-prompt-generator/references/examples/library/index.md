# 案例库索引

本案例库共 100 个校准案例，只服务于封面模式与正文配图模式，不新增第三种运行功能。

## 分片

- `cases-001-025.md`：AI 模型、AI Agent、机器人、芯片、GPU、创业与融资。
- `cases-026-050.md`：商业、安全、教程、工作流与产品测评。
- `cases-051-075.md`：行业分析、公司分析、政策、自动驾驶、医疗 AI 与教育 AI。
- `cases-076-100.md`：设计 AI、办公 AI、未来趋势、赚钱、营销、产品发布与采访。

## 检索方式

先按主题选择分片，再搜索模式、目标模型或场景词。例如搜索 `封面模式`、`Seedream`、
`桌面实物演示`、`自然窗光` 或 `标题留白`。案例编号只用于定位，不代表质量排序。

使用案例时，应比较文章理解、视觉策略和 Prompt 之间的对应关系，再根据当前文章重新
提炼观点与场景。优先检索“相同关系、不同题材”或“相同题材、不同意图”的案例。

## 使用边界

禁止直接复制案例 Prompt，禁止把案例当作固定答案或关键词模板。不得因案例覆盖新闻、
教程、评测等题材而新增运行模式；这些仍然只是文章分类或作者意图。
